Fundamentals of Computer Engineering

Fundamentals of Computer Engineering is a subject offered to double-degree students at UFV. It's worth 3 ECTS and spans one semester, combining theoretical knowledge with practical web development and computing foundations. It is divided into five different blocks covering both the theoretical and the practical sub-sections of this subject. Practical Assignments

Practical Work I

Practical Work I was the first of the pracical tasks sent by Moisés, the FCE teacher. It focused on research about seleted topics to promote teamwork investigation and further learning individually. This project then needed to be presented by three of the group members, whilst the rest had to answer questions delivered afterwards by the teacher.

Practical Work II

Practical Work II is the second practical task sent to FCE students. In this task, it is aimed to learn how to correctly use CSS and HTML by using it to construct a website, in my case, this is my Practical Work II submission! It has many other differences with respect to task number 1, like it being on teams and this one being individually made.
